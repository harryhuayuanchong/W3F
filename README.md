# W3F - A framework for reducing gas costs & streamlining configuration

## Requirements
* Node
* Yarn (Frontend)
* Npm (Contract)
* Git

## Tech stack
* Language - Solidity
* Framework - Hardhat
* Frontend - React.js
* Wallet - Metamask
* Web3 library - Ethers.js
* React Hooks - Wagmi
* Node Providers - QuickNode/Infura/Alchemy/...
